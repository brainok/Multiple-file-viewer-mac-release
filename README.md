# File Pad

Public download repository for File Pad.

## Download

Latest DMG:

https://github.com/brainok/Multiple-file-viewer-mac-release/releases/latest/download/File%20Pad.dmg

## Install

1. Open `File Pad.dmg`.
2. Drag `File Pad.app` to the `Applications` folder.
3. Open File Pad from Applications.

## Notes

- The DMG filename is versionless so the latest download URL stays stable.
- The source code repository is private.
